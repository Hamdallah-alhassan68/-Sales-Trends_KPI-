📊 Sales Trends & KPI Dashboard

An interactive Power BI dashboard designed to analyze sales performance, track key business metrics, and support data-driven decision-making using real-world retail data.

📌 Overview

This project analyzes retail sales data to uncover trends in revenue, profitability, and product performance. The dashboard enables stakeholders to quickly identify growth opportunities and monitor key performance indicators (KPIs).

The dataset used is the Superstore Dataset, a widely used dataset for sales analytics.

🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing products and regions
Track monthly and yearly growth trends
Improve reporting efficiency through automation
✨ Features
📈 Revenue & Profit KPIs with dynamic filtering
🌍 Regional Sales Analysis (performance by location)
📦 Product Category Insights (top and underperforming items)
📅 Time-Series Analysis for trend tracking
🎛️ Interactive Filters & Slicers for user-driven exploration
🛠️ Tools & Technologies
Data Source: Excel (.xls)
Data Cleaning & Preparation: Microsoft Excel
Data Visualization: Microsoft Power BI
Dataset: Superstore Dataset
🧠 Data Processing Workflow
Data Collection
Imported Superstore.xls dataset from Kaggle
Data Cleaning (Excel)
Removed duplicates and handled missing values
Standardized column formats (dates, categories, numeric fields)
Created calculated fields for better analysis
Data Modeling (Power BI)
Built relationships and measures (e.g., total sales, profit margin)
Created calculated columns and KPIs
Dashboard Development
Designed interactive visuals and reports
Added slicers for dynamic filtering
Optimized layout for usability and clarity
📊 Key Insights
Identified high-performing product categories contributing the most revenue
Detected regional disparities in sales and profitability
Revealed seasonal trends affecting sales performance
Highlighted opportunities to improve profit margins
📈 Impact
📉 Reduced reporting time by ~30% through automation
⚡ Enabled faster decision-making with real-time KPI tracking
📊 Transformed raw data into actionable business insights
